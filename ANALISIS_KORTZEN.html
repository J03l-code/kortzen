Análisis Integral del Proyecto KORTZEN

Plataforma de Gestión de Barbería Premium

1. Visión General del Proyecto

KORTZEN no es solo un sistema de reservas; es una plataforma integral diseñada para elevar la experiencia de la barbería tradicional a un nivel de lujo digital. El proyecto fusiona una estética visual impactante ("Dark Premium") con una arquitectura robusta de gestión empresarial.

Objetivo Principal: Ofrecer una experiencia de usuario fluida y sofisticada para el cliente final, mientras proporciona herramientas de gestión potentes y detalladas para administradores y barberos.

---

2. Estética y Experiencia de Usuario (UX/UI)

El diseño sigue una línea "Gentleman's Luxury":

  • Paleta de Colores: Negro Mate, Carbón, y Dorado Premium (#D4AF37) para acentos.
  • Tipografía: Combinación moderna para títulos de impacto y textos legibles.
  • Interacciones:
  • Preloader Cinematográfico: Animación de entrada de marca.
  • Cursor Personalizado: Detalles sutiles que refuerzan la identidad de marca de lujo.
  • Micro-animaciones: Efectos hover, transiciones suaves y reveals al hacer scroll.
  • Hero Video: Fondo de video de alta calidad en la página de inicio para inmersión inmediata.

---

3. Desglose Funcional por Módulos

A. Portal Público (Cliente Final)

El "escaparate" digital de la barbería.

1. Landing Page Dinámica:

  • Carrusel de testimonios.
  • Sección de equipo (Barberos) cargada dinámicamente según la sucursal.
  • Galería de servicios destacados.

2. Sistema "Gentleman's Quiz":

  • Herramienta interactiva que recomienda estilos basados en la rutina y preferencias del usuario.

3. Wizard de Reservas (Paso a Paso):

  • Selección visual de Barbero.
  • Selección de Servicio.
  • Calendario de disponibilidad en tiempo real (bloquea horas ocupadas).
  • Login/Registro simplificado (Google Auth o Email).
  • Confirmación inmediata.

4. Panel del Cliente (`Mi Perfil`):

  • Historial de citas pasadas y futuras.
  • Posibilidad de reagendar o cancelar (según reglas de negocio).
  • Gestión de datos personales.

B. Módulo Administrativo (Back-Office)

El "cerebro" de las operaciones.

1. Dashboard de Inteligencia de Negocios (BI):

  • KPIs en Tiempo Real: Ventas del día, Citas pendientes/completadas.
  • Gráficos Financieros: Tendencia de ventas de los últimos 7 días.
  • Tasa de Retención: Métrica automática de fidelización de clientes.

2. Gestión de Sucursales:

  • Soporte multi-sucursal (ej: Llano Chico, La Mariscal).
  • Configuración independiente de horarios y datos de contacto per sucursal.

3. Gestión de Inventario:

  • Control de stock por sucursal.
  • Sistema de Alertas: Notificaciones visuales cuando un producto está bajo el stock mínimo.
  • Valoración económica del inventario actual.

4. Gestión de Usuarios y Roles:

  • Admin Global: Acceso total.
  • Admin Local (Gerente): Acceso limitado a su sucursal.
  • Barbero: Acceso operativo limitado.

C. Módulo Operativo (Barberos)

Herramientas para el día a día del profesional.

1. Agenda Digital Personal:

  • Visualización clara de "Próximo Cliente" (Next Up Widget).
  • Calendario mensual de ocupación.
  • Detalle diario de citas.

2. Gestión de Disponibilidad:

  • Configuración de horarios de trabajo semanales.
  • Bloqueo de días (Vacaciones/Descansos).

3. Finanzas Personales:

  • Calculadora automática de comisiones (Día/Semana/Mes).
  • Transparencia en sus ingresos generados.

4. Arquitectura Técnica

Stack Tecnológico

  • Backend: PHP 8+ (Optimizado, sin frameworks pesados para máxima velocidad).
  • Base de Datos: MySQL (Relacional, con integridad referencial estricta).
  • Frontend: HTML5, CSS3 (Variables nativas), JavaScript Vainilla (ES6 Modules).
  • Seguridad:
  • Encriptación de contraseñas con BCRYPT.
  • Protección contra inyección SQL (PDO Prepared Statements).
  • Control de sesiones basado en Roles.

Modelo de Datos (Core)

El sistema se basa en 9 entidades principales interconectadas:

1. `sucursales`: Centros operativos.

2. `usuarios`: Personal interno.

3. `clientes`: Usuarios finales (con soporte OAuth Google).

4. `servicios`: Catálogo comercial (precios, duraciones).

5. `inventario`: Stock de productos.

6. `citas`: Núcleo transaccional (relaciona Cliente-Barbero-Servicio-Sucursal).

7. `horarios_barberos`: Disponibilidad recurrente.

8. `dias_bloqueados`: Excepciones de calendario.

9. `logs_actividad`: Auditoría de seguridad.

---

5. Valor Diferencial para el Cliente

1. Automatización: Reduce drásticamente el tiempo de gestión telefónica/WhatsApp.

2. Imagen de Marca: Posiciona a la barbería como un establecimiento de alta gama/lujo.

3. Control Total: Permite a los dueños saber exactamente cuánto ganan, quién trabaja y qué falta en inventario sin estar presentes.

4. Escalabilidad: Listo para abrir nuevas sucursales sin cambiar el software.

---

6. Características Avanzadas y Detalles Técnicos

Para garantizar la integridad y utilidad del sistema a largo plazo:

A. Gestión de Galería (Dynamic Portfolio)

  • Carga de Imágenes: Módulo para subir fotos de cortes recientes.
  • Categorización: Clasificación automática (Corte, Barba, Local, Productos).
  • Control Total: Interfaz para eliminar o actualizar fotos que se muestran en la landing pública.

B. CRM y Fidelización (Gestión de Clientes)

  • Integración WhatsApp: Botón directo en el perfil del cliente para iniciar chat sin guardar contacto.
  • Notas Privadas: Sistema de notas internas para que el barbero recuerde preferencias del cliente (ej. "Usa cera mate", "Prefiere no hablar mucho").
  • Métricas de Cliente: Cálculo automático del "Valor de Vida" (LTV) del cliente (Total gastado histórico).

C. Seguridad y Auditoría

  • Logs de Actividad: Registro inmutable de acciones críticas (Quién borró una cita, quién modificó inventario).
  • Traza de IP: Registro de direcciones IP y navegadores para detectar accesos sospechosos.

D. Lógica de Negocio Automática

  • Dedúcción de Inventario: Al completar una cita, el sistema puede descontar automáticamente los insumos utilizados (Feature ready).
  • Validación de Horarios: Algoritmo que cruza horarios de sucursal, horarios del barbero, días bloqueados y citas existentes para evitar overbooking.
